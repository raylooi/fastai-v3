Classify Waste for Sorting

Abstract: Training was done on a set of 2527 images split into 2022 for training (80%) and 505 for validation (20%). Object classes: 'cardboard', 'glass', 'metal', 'paper', 'plastic', 'trash'.
(Dataset can be downloaded from -->[here].)

Transfer learning was first done using a pretrained ResNet50 as backbone and a single hidden layer as classifier. With the initial layers frozen, weights were learnt using the last fully connected layers.
After this, the layers were 'unfreezed' so the entire model was trained.

Final training loss = 0.128775 and error rate=0.071287 (93% accuracy) after 20 epochs.
Total training time was about 30 minutes using ASUS GL703GS-E5010T with GTX 1070 GPU card.


Use images of 'cardboard', 'glass', 'metal', 'paper', 'plastic' or 'trash'
